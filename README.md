# Pong Game

A classic Pong game implementation using HTML5 Canvas and JavaScript.

## How to Play

1. Open `index.html` in your web browser
2. Press **SPACE** to start the game
3. Use the following controls:
   - **Player 1 (Left Paddle)**: W (up) / S (down)
   - **Player 2 (Right Paddle)**: ↑ (up) / ↓ (down)
4. Press **SPACE** to pause/resume the game

## Game Features

- Two-player gameplay
- Ball physics with collision detection
- Score tracking
- Responsive paddle controls
- Retro green-on-black visual style
- Pause/resume functionality

## Files

- `index.html` - Main game page
- `pong.js` - Game logic and mechanics
- `style.css` - Game styling and layout

## Running the Game

Simply open the `index.html` file in any modern web browser. No additional setup or dependencies required!
